# DailyWorkerApp

Unified repository for the DailyWorker project.

## Project Structure

- **/WorkerBackend**: Spring Boot Java application handling business logic, user management, and job matching.
- **/WorkerD**: React Native application for Workers and Hirers.

## Backend Features
- Redis-based caching
- JWT Authentication
- Rate Limiting
- OpenCV/OpenSearch integration (planned)

## Mobile Features
- Location-based job search
- Real-time notifications
- Multilingual support
